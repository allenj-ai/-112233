后台 PRD 文档说明
1. 市场管理 - 统计区字段说明
市场总数：当前筛选条件下的市场数量（单位：个）。
总交易量：当前筛选结果内所有市场累计成交金额（单位：USDT）。
累计手续费：当前筛选结果内所有市场累计手续费（单位：USDT）。
交易参与人数：当前筛选结果内交易人数合计（展示口径：人次）。
2. 市场管理 - 市场列表字段说明
市场：展示市场标题；副信息展示 Market ID 与上架状态（已上架/已下架）。
状态：进行中、已封盘、结算中、公示期、争议中、已结束。
订单模式：AMM 或 订单簿。
总交易量：该市场累计成交金额（USDT）。
总手续费：该市场累计手续费（USDT）。
交易人数：该市场累计交易参与人数（人次）。
总流动性：该市场当前资金池规模（USDT）。
兑付金额：该市场结算需兑付给用户的金额（USDT）。
LP 分配金额：总流动性 - 兑付金额（USDT）。
封盘时间：编辑弹窗字段，停止新下单的时间点（yyyy-mm-dd hh:mm）。
结束时间：市场交易结束时间（yyyy-mm-dd hh:mm）。
3. 市场管理 - 操作与筛选说明
搜索框：按市场标题关键词模糊匹配。
状态筛选：按状态过滤列表与统计区数据。
编辑：仅可修改市场标题、市场描述、封盘时间、结束时间。
下架/恢复上架：切换市场展示状态。
限制：市场管理中无法直接改变市场状态，也不提供“提前结束”操作；状态变更仅能在结算管理执行。
4. 结算管理 - 参数配置字段说明
提案保证金（USDT）：发起提案时需缴纳的保证金，默认 500。
争议保证金（USDT）：发起争议时需缴纳的保证金，默认 500。
公示期（小时）：结算公示持续时间，默认 24h。
保存参数：提交并生效上述配置，同时写入操作记录。
5. 结算管理 - 结算列表字段与操作
列表范围：展示全部市场（含进行中市场）。
展示字段：市场、状态、订单模式、总流动性、兑付金额、LP 分配金额、结束时间。
提前结束：唯一可执行动作，进行中/已封盘/结算中/公示期/争议中市场均可执行；执行后状态改为“已结束”，结束时间更新为当前时间。
6. 结算管理 - 操作记录字段说明
记录时间：操作发生时间（yyyy-mm-dd hh:mm）。
记录内容：操作类型 + 市场 ID 或参数值变更内容。
记录范围：参数保存、提前结束、上下架、市场编辑。
展示顺序：最新记录在最上方。
